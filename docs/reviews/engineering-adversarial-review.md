## 已验证边界

| 严重度 | 风险 | 验证与处置 |
| --- | --- | --- |
| P0 | 地图或模型密钥进入仓库 | Git 扫描无真实密钥；`.env.local` 被忽略；Web Service 与 JS 安全密钥仅在服务端 |
| P0 | 高德 JS 安全密钥明文暴露 | 按官方推荐改为 `/_AMapService` 服务端代理；页面仅包含公开 JS Key |
| P0 | 分片余额重复领取或变负 | reducer 与仓库动作 ID 双重幂等；消费和物品发放在同一状态事务 |
| P0 | 地图随机碎片可被重复刷接口套利 | `CLAIM_MAP_SHARD` 使用 `map-shard:${shardId}` 动作 ID，仓库和 reducer 均幂等；输入金额限制为 1–5 |
| P0 | 登录 Cookie 可伪造 | 随机 256 位令牌，服务端仅保存 SHA-256；HttpOnly、SameSite=Lax、生产 Secure |
| P1 | Coze 输出注入或失控 | 严格 JSON 字段、候选锚点白名单、控制字符清理、72 字上限、8 秒超时 |
| P1 | 路线泄露住址/目的地 | 完成同行时移除首尾路线段；训练路线不作为真实记录 |
| P1 | 本地开发模块读取陈旧状态 | E2E 复现后将仓库实例提升为进程级唯一实例；PostgreSQL 生产路径仍使用事务锁 |
| P1 | Next 预取返回结算前库存 | 状态敏感导航设置 `prefetch={false}` |
| P2 | 地图服务中断 | Imagegen 成都图与训练锚点降级；不生成真实打卡 |
| P2 | 定位权限被拒后误切成假真实地图 | 定位失败只显示“定位未授权或暂时不可用”，不生成真实到访；高德底图仍可浏览 |
| P2 | 移动端断链或溢出 | 390×844 手动验收，桌面与手机 Chromium E2E 均通过 |
